# Phase 1: メタ学習器（Meta-learners）によるアップリフトモデリングの探求

## 1. プロジェクト概要
本プロジェクトでは、因果推論の機械学習手法である「アップリフトモデリング」を用いて、マーケティング施策（メール送信）の純増効果（Causal Effect）を個人レベルで予測する手法を学習しました。
既存の学習アルゴリズムを組み合わせて因果効果を導き出す「メタ学習器（Meta-learners）」の代表的な3手法を実装し、それぞれの特性と実力値を定量的に比較検証しました。

## 2. 使用データセット
- **データセット名**: Hillstrom Dataset (MineThatData E-Mail Analytics)
- **概要**: Eコマースにおける顧客64,000人に対するメールキャンペーンのA/Bテストデータ。
- **介入 (Treatment)**: 
  - `1`: 男性向けメール送信 (Mens Email)
  - `0`: メール送信なし (Control)
- **ターゲット (Outcome)**: `spend` (購入金額)

## 3. 実装アルゴリズムの解説

### ■ S-Learner (Single-Learner)
- **仕組み**: 介入変数（メールの有無）を、他の特徴量（過去の購入額等）と同様に「ただの1つの特徴量」として扱い、単一のモデルで学習する手法。
- **メリット**: データを分割せずに全量学習できるため、データ効率が良い。
- **デメリット**: 介入変数の影響力が他の特徴量に比べて小さい場合、モデルに無視されやすく、アップリフト（差分）が正確に算出されない「ゼロの過剰発生」が起きやすい。

### ■ T-Learner (Two-Learner)
- **仕組み**: 介入群と統制群でデータを二分割し、それぞれ独立した2つのモデルを構築する手法。
- **メリット**: 介入の有無による差異を確実に捉えることができる。
- **デメリット**: 学習に使えるデータが群ごとに半分になるため、データ量が少ない場合に過学習や精度の低下を招きやすい。

### ■ X-Learner
- **仕組み**: T-Learnerを拡張した3段階の手法。
  1. ベースモデルで「もしもの世界」を予測。
  2. 現実と予測の差分から「仮想的な因果効果（D）」を算出。
  3. その「効果そのもの」を直接予測するモデルを構築し、最後に重み付け統合する。
- **メリット**: 介入群と統制群の人数に極端な偏りがある「不均衡データ」に非常に強い。

## 4. 定量評価指標: AUUC
モデルが「説得可能層（介入すれば反応する人）」をどれだけ正確に優先順位付けできているかを測る指標として **AUUC (Area Under the Uplift Curve)** を採用しました。

- **Random Baseline**: ターゲットをランダムに選んだ場合の基準値。これを超えているかどうかが、AI導入の価値を判断するボーダーラインとなる。

## 5. 実験結果と考察

### ■ AUUC スコアボード
| 順位 | モデル | AUUC スコア | 備考 |
| :--- | :--- | :--- | :--- |
| **1位** | **T-Learner** | **2,131,879,738** | 今回のベストモデル |
| 2位 | X-Learner | 2,001,649,215 | 僅差で2位 |
| 3位 | S-Learner | 1,791,914,151 | 特徴量無視の傾向あり |
| - | Random | 590,221,764 | 比較基準値 |

### ■ 総括
理論上の汎用性が高いとされるX-LearnerがT-Learnerに敗退するという、非常に示唆に富む結果となった。
原因として、今回のデータセットが**「A/Bテストにより群の人数が均等であったこと」**、および**「ノイズの多いビジネスデータにおいて、モデルを多層化するX-Learnerの誤差蓄積が影響したこと」**が推察される。

この実験を通じ、データサイエンスにおいては「常に最強のモデル」は存在せず、**「ランダムベースラインとの比較」**および**「複数のメタ学習器によるトーナメント方式の評価」**が実務上の意思決定において極めて重要であることを学んだ。
